# daspots-frontend

> Frontend for DaSpots.com

## About

This project uses [React Native](https://facebook.github.io/react-native/) that lets you build apps for iOS/Android. In this project we use [kanzitelli/react-native-navigation-mobx-boilerplate](https://github.com/kanzitelli/react-native-navigation-mobx-boilerplate) as starting point.

## Getting Started

1. Make sure you have [NodeJS](https://nodejs.org/), [npm](https://www.npmjs.com/) and [React Native](https://facebook.github.io/react-native/docs/getting-started.html) installed.
2. Clone this repo: `git clone https://github.com/kanzitelli/daspots-frontend.git`
3. Install all dependencies: `cd daspots-frontend && npm i -S`
4. Make sure that `rethinkdb` and `npm start` are running in terminal.
5. Make sure you have `./config` directory in the project. (if not, it should be in Slack in development channel)
6. `react-native run-ios` or `react-native run-android`
